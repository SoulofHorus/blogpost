Step One: Making a GitHub page/repository

first of all, if you don't have a GitHub account make one. It is super easy. Then, make a respository labeled 'blogpost' or something similar. Once that is done, add an index.html file. Then, install Atom with Teletype so that when you type, auto complete can do it for you. Trust me, it is a life-saver.

From here, you have to make sure you commit often, and make sure to push to origin. Save contantly.

Step Three: What Now?

Now that you are looking at the coding window, it asks you for your user name. Your username should be for the first part of your email. For example, mine is 'hammonkm'. Then using the IP address provided on Moodle, you can log in.

Once that is done, using the log in code: $ ssh username@64.227.19.180, where username is the first part of your Clarkson email.

Step Four: Making a folder

To get your project on the web, you'll need to set up a directory inside the web root. To change from your initial working directory to the web root, run the following command: $ cd ../../var/www/html.

Then, you can make your project directory inside the html folder. Please name it your username. You can put any number of other subdirectories inside it.

Step Five: You're Done!

Congratulations, you are done! Now you can use this folder to do whatever you like. Please be sure to use the command 'sudo' to change the permissions so you can access your own folder and make changes!
